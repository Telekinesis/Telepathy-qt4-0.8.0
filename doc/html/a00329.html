Detailed Description

Mapping type generated from the specification. Convertible with QMap<QString, QVariantMap>, but needed to have a discrete type in the Qt4 type system.

A map from protocol identifiers supported by a connection manager to the immutable properties of the corresponding ProtocolInterface objects.
